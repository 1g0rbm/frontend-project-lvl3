### Installation
```bash
git clone git@github.com:1g0rbm/frontend-project-lvl3.git;
cd frontend-project-lvl3

# build with develop mode
make dev
# build with production mode
make build
# run dev server
make start
# run liners, stylecheck and tests
make check
```
